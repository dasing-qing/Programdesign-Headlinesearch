# 简易版头条搜索
### 简单实现了头条搜索的部分功能
## 功能实现
一个简易的头条搜索，实现搜索框、搜索结果页列表，实现swiper对搜索结果页切换，而且有部分动画效果支持页数导航等引入了particles、sweetalert2两个插件丰富页面展现交互效果，提升用户体验
## 技术选型
### 前端技术选型
### jQuery
